RESEARCH TOOLKIT

把研究方法
變成可以動手試的小工具

以下工具均依據公開發表的學術文獻搭建，覆蓋公共衛生、健康行為與研究方法等方向。 所有運算在使用者的瀏覽器內完成，數據不離開本地設備，不收集、不儲存、不上傳任何個人資料， 僅作為學習交流與資訊參考之用。

支援中文名稱、英文標籤、工具描述的即時模糊匹配
CATEGORY 01研究設計與規劃 CATEGORY 02數據處理與分析 CATEGORY 03系統綜述與報告 CATEGORY 04健康評估與管理 CATEGORY 05學術發展與職涯

CATEGORY 01

研究設計與規劃

4 TOOLS
研究啟動前的必要準備，包含文獻檢索策略、倫理申請文件、統計方法選擇與樣本量估算。 覆蓋從選題、研究設計、方法決策到資源規劃的整體前期流程。
01Search

文獻檢索式生成器

依據 PICO、SPIDER 等常見檢索框架，組合關鍵詞與布林邏輯， 生成可用於 PubMed、Web of Science 等資料庫的檢索式草稿。

Literature Review Search Strategy
OPEN TOOL
02Ethics

研究倫理申請文件生成器

按照常見的學術倫理審查框架，引導填寫研究背景、設計方法、知情同意與 隱私保護等要素，輔助快速搭建一份結構完整的倫理申請草稿。

Research Methods Document Tool
OPEN TOOL
03Method Selector

統計方法選擇器

以決策樹的方式，根據研究問題、變項類型、組數與分布假設， 推薦合適的統計檢驗方法（t 檢驗、ANOVA、卡方、回歸等），並附帶適用條件說明。

Study Design Decision Tree
OPEN TOOL
04Sample Size

樣本量計算器

依據假設檢驗類型、預期效應量、檢驗效能與顯著性水平，計算研究設計所需的最小樣本量， 涵蓋均值比較、率比較與相關性分析等常見研究情境。

Study Design Calculator
OPEN TOOL
CATEGORY 02

數據處理與分析

5 TOOLS
收集數據之後的核心環節，依序處理數據清洗、隱私保護、量表信度檢驗、診斷效能評估與 效應量轉換。所有運算皆在瀏覽器本地完成，原始數據不會離開使用者的設備。
05Data Cleaning

缺失值與異常值處理助手

針對上傳的數據集，識別缺失值的分布類型（MCAR / MAR / MNAR）與異常值候選， 並依據變項特性推薦插補方法與處理策略。

Data Preprocessing Helper Tool
OPEN TOOL
06Privacy

匿名化工具生成器

為含個人資訊的研究數據提供匿名化方案參考，依據常見的隱私保護準則， 輔助識別敏感欄位並生成相應的去識別化處理建議與作業流程。

Data Privacy Reference
OPEN TOOL
07Reliability

問卷信度分析器

輸入問卷或量表的原始作答數據，計算 Cronbach's α、刪題後 α 等信度指標， 輔助量表開發與內部一致性檢驗，並提供分項目的描述性統計。

Psychometrics Analysis Tool
OPEN TOOL
08Diagnostic

ROC / AUC 與最佳截斷值計算器

輸入連續型診斷指標與金標準分類數據，繪製 ROC 曲線並計算 AUC， 依據 Youden 指數推薦最佳截斷值，並輸出對應的敏感度、特異度與預測值。

Diagnostic Test Calculator
OPEN TOOL
09Effect Size

效應量轉換器

支援 Cohen's d、Hedges' g、相關係數 r、Odds Ratio 等常見效應量之間的相互轉換， 方便整合異質性研究結果，輔助 meta 分析與證據綜合。

Meta-analysis Converter
OPEN TOOL
CATEGORY 03

系統綜述與報告規範

3 TOOLS
研究撰寫與投稿前的最後一公里。覆蓋系統綜述流程圖繪製、報告清單核對、 以及證據質量評級，協助提升綜述類研究的透明度與報告完整性。
10Systematic Review

PRISMA 流程圖生成器

依據 PRISMA 2020 規範，依序填入文獻識別、去重、篩選與納入排除的數量， 自動生成可下載的系統綜述流程圖。

Evidence Synthesis Diagram Tool
OPEN TOOL
11Checklist

研究報告清單檢查器

內建 CONSORT、STROBE、PRISMA 等常見研究報告規範清單， 逐條核對研究稿件的覆蓋情況，輔助投稿前自查與報告完整性提升。

Reporting Standards Quality Tool
OPEN TOOL
12Evidence Quality

GRADE 證據質量摘要表生成器

按照 GRADE 工作組的方法學框架，協助對結局指標逐項評估證據質量， 生成結構化的證據摘要表（Summary of Findings）。

Evidence Synthesis Quality Rating
OPEN TOOL
CATEGORY 04

健康評估與管理

6 TOOLS
將研究方法落地到健康應用場景：先做全面健康評估與風險篩查，再到素養測量、個性化處方， 最後延伸到健康服務與管理、特定情境的健康參考。圍繞個體與服務端的整體流程展開。
13Assessment

標準全面健康評估工具

整合體格、生活方式、心理與慢性病風險等多維度問項， 生成一份結構完整的綜合健康評估報告，輔助體檢解讀與健康管理規劃。

Comprehensive Assessment Tool
OPEN TOOL
14Cardiovascular

ASCVD 心血管疾病風險計算器

依據 ACC/AHA 匯總隊列方程，估算 40–79 歲人群未來 10 年發生 動脈粥樣硬化性心血管疾病（ASCVD）的概率，輔助一級預防決策思考。

Public Health Risk Calculator
OPEN TOOL
15Literacy

健康素養評估

依據國際通用的健康素養量表，評估個體在獲取、理解、評估與運用健康資訊 方面的綜合能力，並提供分維度反饋。

Health Behavior Self-assessment
OPEN TOOL
16Prescription

標準健康處方工具

依據評估結果與標準化的健康處方框架，圍繞運動、飲食、作息與行為干預等方向， 生成個性化的健康處方建議。

Lifestyle Prescription Tool
OPEN TOOL
17Service & Management

健康服務與管理工具包

面向健康服務設計與管理場景，整合常用的服務流程、品質指標與管理框架， 協助梳理健康項目的規劃、執行與評估，作為服務端的綜合工具參考。

Health Management Toolkit
OPEN TOOL
18Travel

國際旅遊健康參考

整合公開的國際旅遊健康資訊，依目的地與行程特徵， 梳理常見的健康風險、疫苗接種建議與基本注意事項，作為出行前的資訊參考。

Travel Health Reference
OPEN TOOL
CATEGORY 05

學術發展與職涯

2 TOOLS
面向研究生與青年研究者，提供階段性的成長路徑梳理與學科趨勢視角， 作為長期學術規劃與選題方向的補充參考。
19Growth

研究生成長規劃工具

為研究生提供階段性成長規劃參考，從研究方向確立、論文寫作、學術發表 到職涯路徑，結構化梳理常見里程碑與學習資源建議。

Career Planning Reference
OPEN TOOL
20Trends

學術研究趨勢

梳理近年來大健康、公共衛生、酒店與款待、旅遊等領域學術研究的熱點議題， 為相關方向的選題、文獻綜述與課堂教學提供參考視角。

Hospitality Literature Insights
