RESEARCH TOOLKIT

把研究方法
變成可以動手試的小工具

以下工具均依據公開發表的學術文獻搭建，覆蓋公共衛生、健康行為與研究方法等方向。 所有運算在使用者的瀏覽器內完成，數據不離開本地設備，不收集、不儲存、不上傳任何個人資料， 僅作為學習交流與資訊參考之用。

CATEGORY 01

健康評估工具

基於國際指南與標準量表搭建的線上計算器，協助理解個體健康風險、健康素養水平， 以及國際旅行中可能涉及的健康注意事項。所有計算邏輯皆在頁面註明出處。
01 Cardiovascular

ASCVD 動脈粥樣硬化性心血管疾病風險計算器

依據 ACC/AHA 匯總隊列方程，估算 40–79 歲人群未來 10 年發生 動脈粥樣硬化性心血管疾病（ASCVD）的概率，輔助一級預防決策思考。

Public Health Risk Calculator
OPEN TOOL
02 Literacy

健康素養評估

依據國際通用的健康素養量表，評估個體在獲取、理解、評估與運用健康資訊 方面的綜合能力，並提供分維度的反饋，方便個人或教學場景使用。

Health Behavior Self-assessment
OPEN TOOL
03 Travel

國際旅遊健康參考

整合公開的國際旅遊健康資訊，依目的地與行程特徵， 梳理常見的健康風險、疫苗接種建議與基本注意事項，作為出行前的資訊參考。

Travel Health Reference
OPEN TOOL
CATEGORY 02

研究流程與規劃工具

面向研究流程中的常見場景，從倫理申請、文獻檢索，到研究生階段的成長規劃與學科趨勢梳理， 提供一些可以動手嘗試的小工具，希望讓研究方法的學習與實踐更直觀一些。
04 Ethics

研究倫理申請文件生成器

按照常見的學術倫理審查框架，引導填寫研究背景、設計方法、知情同意與 隱私保護等要素，輔助快速搭建一份結構完整的倫理申請草稿。

Research Methods Document Tool
OPEN TOOL
05 Search

文獻檢索式生成器

依據 PICO、SPIDER 等常見的檢索框架，協助組合關鍵詞與布林邏輯， 生成可用於 PubMed、Web of Science 等資料庫的檢索式草稿，提升文獻檢索效率。

Literature Review Search Strategy
OPEN TOOL
06 Growth

研究生成長規劃工具

為研究生提供階段性的成長規劃參考，從研究方向確立、論文寫作、學術發表 到職涯路徑，結構化梳理常見里程碑與相應的學習資源建議。

Career Planning Reference
OPEN TOOL
07 Trends

學術研究趨勢

梳理近年來大健康、公共衛生、酒店與款待、旅遊等領域學術研究的熱點議題， 為相關方向的選題、文獻綜述與課堂教學提供一個可參考的視角。

Hospitality Literature Insights
OPEN TOOL
CATEGORY 03

統計與數據工具

覆蓋研究設計與數據處理階段的常見任務，包括樣本量估算、量表信度檢驗， 以及含個人資訊資料的匿名化處理。所有運算皆在瀏覽器本地完成，原始數據不會離開使用者的設備。
08 Statistics

樣本量計算器

依據假設檢驗類型、預期效應量、檢驗效能與顯著性水平，計算研究設計所需的最小樣本量， 涵蓋均值比較、率比較與相關性分析等常見研究情境。

Study Design Calculator
OPEN TOOL
09 Reliability

問卷信度分析器

輸入問卷或量表的原始作答數據，計算 Cronbach's α、刪題後 α 等信度指標， 輔助量表開發與內部一致性檢驗，並提供分項目的描述性統計。

Psychometrics Analysis Tool
OPEN TOOL
10 Privacy

匿名化工具生成器

為含個人資訊的研究數據提供匿名化方案參考，依據常見的隱私保護準則， 輔助識別敏感欄位並生成相應的去識別化處理建議與作業流程。

Data Privacy Reference
OPEN TOOL
USAGE NOTE

以上工具均為個人學術交流用途，內容僅作資訊參考與學習之用， 不構成醫療、法律或正式學術建議。具體應用請結合專業判斷或諮詢相關領域人員。
