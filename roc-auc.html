DIAGNOSTIC · BIOSTATISTICS

ROC / AUC 與最佳截斷值計算器

將真實結局（0 / 1）與對應的預測分數或檢測值貼上，工具會自動繪製 ROC 曲線、 計算 AUC，並依 Youden 指數搜尋最佳截斷值，同時輸出敏感度、特異度、 陽性預測值（PPV）與陰性預測值（NPV）。所有計算均在瀏覽器本地完成， 資料不會上傳。

📥 CSV 文本貼入 📈 ROC 曲線繪製 🎯 Youden 最佳截斷 🔒 本地計算 ⬇ 圖表 / 表格匯出
PART 01 · DATA INPUT

輸入資料

支援兩種貼入方式，二擇一即可。資料量越大，曲線越平滑。

資料格式說明　
・方式一（兩欄分別貼入）：真實結局 為 0 / 1（1 代表陽性／患病）， 預測分數 為任意連續數值（如機率、生物標誌物濃度、模型輸出 logit 等）。
・方式二（CSV 貼入）：每列一筆樣本，欄位以逗號 / 分號 / Tab / 空白分隔， 支援含表頭的兩欄資料，例如 label,score。
・小數點請用 .，列數需一致。空白與非數值列會被自動忽略。
貼入 CSV 後可直接點「開始分析」，工具會優先解析 CSV。
