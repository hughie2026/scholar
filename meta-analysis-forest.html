Meta-Analysis Forest Plot

手動輸入多項研究的效應量與不確定度，本工具會自動完成倒方差合併、異質性檢驗， 並繪製簡易森林圖。支援 OR / RR / HR / MD / SMD 五種效應量， 提供固定效應與 DerSimonian-Laird 隨機效應兩種模型。所有計算在瀏覽器本地進行。

📐 Inverse-Variance 🎲 DerSimonian-Laird 🌲 Forest Plot 🔒 本地計算 📄 PNG / CSV 匯出
教學與初步分析工具　 本工具用於統計教學、概念演示與初步探索，不替代 RevMan、Stata、R meta / metafor 等正式 Meta 分析軟體。 正式發表的系統綜述與 Meta 分析請使用標準工具，並遵循 PRISMA、Cochrane Handbook 等指引。
PART 01

分析設置

選擇效應量類型與資料輸入方式。比值型效應量 (OR / RR / HR) 將在對數尺度上合併。

METHODS

方法與公式

  • 固定效應採用倒方差加權法：wᵢ = 1/SEᵢ²，θ̂ = Σ(wᵢθᵢ)/Σwᵢ。
  • 異質性 Q = Σwᵢ(θᵢ − θ̂)²，I² = max(0, (Q − df)/Q) × 100%。
  • 隨機效應採用 DerSimonian-Laird 估計：τ² = max(0, (Q − df)/C)，C = Σwᵢ − Σwᵢ²/Σwᵢ。
  • OR / RR / HR 在自然對數尺度上合併與檢驗，最後轉回原尺度。
  • Z = θ̂/SE(θ̂)，P = 2 × (1 − Φ(|Z|))。
免責聲明　 本工具僅用於統計教學、概念演示與初步探索分析，計算結果不構成正式 Meta 分析或臨床決策依據。 正式系統綜述與 Meta 分析請使用 RevMan、Stata、R meta / metafor 等專業軟體， 並遵循 PRISMA 2020、Cochrane Handbook 等規範。
